# RegistroBasicoDeVentasDiarias
El objetivo de este repositorio es crear una herramienta para RiwiTechStore la cual pueda registrar y validar las ventas diarias de forma estructurada.

# prerequisitos
para ejecutar este proyecto necesitas tener instalado python.

Tambien debes tener un editor de codigo como visual studio o pycharm
